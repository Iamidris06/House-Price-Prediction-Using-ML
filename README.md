# House-Price-Prediction-Using-ML
This project predicts house prices using machine learning.

## Features
- Data preprocessing
- Data visualization using matplotlib
- Regression model for prediction

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## How to Run
1. Install required libraries
2. Run the Python file

## Project Overview

The House Price Prediction project is a machine learning-based application designed to estimate the prices of houses based on various input features such as location, area, number of rooms, and other relevant factors.

The main objective of this project is to understand how data-driven models can be used to solve real-world problems. The project involves collecting and preprocessing data, analyzing patterns through visualization, and applying regression techniques to build a predictive model.

The workflow of the project includes:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA) using visualization tools
- Feature selection and preparation
- Model training using regression algorithms
- Model evaluation using training and testing data

This project helped in gaining practical exposure to machine learning concepts, data handling, and model evaluation techniques.
